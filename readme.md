## How It Works
- Cat data is fetched dynamically from a public REST API
- Users can save cats, which are stored using browser local storage
- Saved cats persist across sessions and are displayed on the ‚ÄúMy Cats‚Äù page
- Breed-based filtering allows users to explore cats by category
